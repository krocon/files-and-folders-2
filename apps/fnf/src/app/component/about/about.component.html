Files and Folders (FnF)

Become a master of file management with "Files and Folders" (FnF).

FnF is an orthodox browser based file manager for Mac OS X, Windows and Linux & Unraid.

The server component based on Node.js (Nest.JS/Express), the client component on Angular.

It's an early version! Tested on Windows 10 and MacOS. Documentation will be expanded and new features will be added.
